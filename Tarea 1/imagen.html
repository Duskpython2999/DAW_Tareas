Etiquetas de estructura de Imagen:

Etiqueta <img>

La etiqueta <img> define una imagen, que puede tener un atributo src para indicar la URL de la imagen, un atributo alt para indicar el texto alternativo, y otros atributos para el tamaño, título o estilo.


Etiqueta <picture>

La etiqueta <picture> define una imagen con varias fuentes, que puede contener elementos <source> para indicar diferentes versiones según el tamaño de la pantalla, el tipo de dispositivo o el formato de la imagen.


Etiqueta <source>

La etiqueta <source> define una fuente de imagen o audio dentro de un elemento <picture> o <audio>, con atributos srcset y media para especificar la URL y la condición de la fuente.

Etiqueta <canvas>

La etiqueta <canvas> define un lienzo, que puede contener gráficos dibujados con JavaScript o con la API de WebGL.


Etiqueta <svg>

La etiqueta <svg> define un gráfico vectorial escalable, que puede contener elementos que definen formas, textos, filtros, animaciones o transformaciones.
