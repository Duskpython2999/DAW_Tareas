Etiquetas de estructura de Imagen:

Etiqueta <audio>

La etiqueta <audio> se utiliza para insertar archivos de audio en una página web. Puedes usarla para reproducir archivos de audio con controles básicos.


Etiqueta <video>

La etiqueta <video> se utiliza para insertar archivos de video en una página web. Puedes incluir múltiples fuentes y controles para la reproducción.


Etiqueta <source>

La etiqueta <source> se utiliza dentro de las etiquetas <audio> y <video> para especificar las fuentes del archivo multimedia en diferentes formatos.


se incluye en los ejemplos de video y audio

Etiqueta <track>

La etiqueta <track> se utiliza dentro de la etiqueta <video> para incluir subtítulos o descripciones de audio.


Etiqueta <canvas>

La etiqueta <canvas> se usa para renderizar gráficos, incluidos los generados dinámicamente, como juegos y visualizaciones.


Etiqueta <svg>

La etiqueta <svg> es un contenedor para gráficos SVG, que tienen varios métodos para dibujar trazados, cuadros, círculos, texto e imágenes gráficas.


Etiqueta <iframe>

La etiqueta <iframe> se utiliza para mostrar contenido de otra fuente dentro de una página web, como un video de YouTube.
