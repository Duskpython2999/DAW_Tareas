Etiquetas de estructura Textos:


Etiqueta <p>

La Etiqueta <p> se utilizan para mostras parrafos, en ellas se pueden mostrar tanto imagenes como enlaces como en el siguiente ejemplo con ayuda de la etiqueta <a>

Puedes incluir otros elementos como enlaces o imágenes dentro de un párrafo.

Etiqueta <br>

La etiqueta <br> define un salto de línea en un documento HTML, que se utiliza para separar el texto en líneas sin crear un nuevo párrafo.

Este es un ejemplo de un texto
con un salto de línea que no crea un nuevo párrafo.
Este texto continúa en la misma estructura, pero en líneas diferentes.

Los saltos de línea
son útiles en ciertas situaciones,
como direcciones o poemas.

Etiqueta <pre>

La etiqueta <pre>define un texto preformateado que conserva los espacios y saltos de línea tal como se escriben en el código fuente.

    Texto preformateado
        - Se respetan los espacios
        - Se respetan los saltos de línea

Etiqueta <blockquote>

La etiqueta <blockquote> define una cita en bloque, con una sangría respecto al texto normal.

"El éxito es la suma de pequeños esfuerzos repetidos día tras día." – Robert Collier

Etiqueta <q>

La etiqueta <q>define una cita en línea, mostrando el texto citado entre comillas.

Albert Einstein dijo: La imaginación es más importante que el conocimiento.


Etiqueta<cite>

La etiqueta <cite>define el título de una obra citada, como un libro, un artículo, o una fuente.

Mi libro favorito es El Principito de Antoine de Saint-Exupéry.


Etiqueta<abbr>

Explicación: La etiqueta <abbr> define una abreviatura o acrónimo, mostrando el significado completo al pasar el cursor sobre el texto.

La ONU tiene varias iniciativas a nivel mundial.


Ejemplo de <dfn>

La etiqueta <dfn>define una definición de un término, que puede mostrar una descripción más detallada al pasar el cursor sobre el texto.

La palabra HTML es el lenguaje utilizado para crear páginas web.


Etiqueta <code>

La etiqueta <code> define un fragmento de código, mostrándolo con una fuente monoespaciada y sin formato especial.

Ejemplo de código: const x = 10;

Etiqueta <kbd>

La etiqueta <kbd> define una entrada de teclado, mostrándola con una fuente monoespaciada y utilizada para representar combinaciones de teclas o comandos.

Presiona Ctrl + C para copiar.


Etiqueta <b>

La etiqueta <b> define un texto en negrita, utilizado para resaltar o enfatizar una parte del texto.

Este es un texto en negrita.


Etiqueta <strong>

La etiqueta <strong> define un texto fuerte, indicando la importancia o la urgencia de una parte del texto.

Este texto es importante.


Etiqueta <i>

La etiqueta <i> define un texto en cursiva, utilizado para marcar una parte del texto como diferente o especial.

Este texto está en cursiva.


Etiqueta <em>

La etiqueta <em> define un texto enfático, utilizado para expresar el énfasis o el tono de una parte del texto.

Este texto está enfáticamente resaltado.


Etiqueta <mark>

La etiqueta <mark> define un texto marcado, utilizado para resaltar o señalar una parte del texto.

Este texto está resaltado.


Etiqueta <small>

La etiqueta <small> define un texto pequeño, utilizado para mostrar información secundaria.

Nota al pie: Este es un texto pequeño.


Etiqueta <del>

La etiqueta <del> define un texto eliminado, mostrándolo con una línea que lo atraviesa.

Texto borrado: Este texto ha sido eliminado.


Etiqueta <ins>

La etiqueta <ins> define un texto insertado, mostrándolo con una línea que lo subraya.

Texto añadido: Este texto ha sido añadido.


Etiqueta <sub>

La etiqueta <sub> define un texto subíndice, que aparece ligeramente por debajo de la línea base.

Fórmula química: H2O


Etiqueta <sup>

La etiqueta <sup> define un texto superíndice, que aparece ligeramente por encima de la línea base.

Exponente: x2


Etiqueta <time>

La etiqueta <time> define una fecha o una hora con un formato estándar o personalizado.

Fecha: 16 de septiembre de 2024


Etiqueta <data>

La etiqueta <data> define un dato con un valor asociado, útil para mostrar información legible por máquinas o humanos.

Valor de temperatura: 23°C


Etiqueta <span>

La etiqueta <span> define una sección de texto, utilizada para aplicar un estilo o manipulación con JavaScript.

Texto en rojo dentro de un párrafo.
