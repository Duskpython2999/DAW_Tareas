Etiquetas de estructura de Enlace:

Etiqueta <a>

La etiqueta <a> define un enlace, que puede contener texto, imágenes u otros elementos.

Visita el sitio web de Example.com para más información.


Etiqueta <area>

La etiqueta <area> define un área de enlace dentro de un elemento <map>, que puede tener forma de rectángulo, círculo o polígono.

Haz clic en la imagen para visitar diferentes secciones:


Etiqueta <map>

La etiqueta <map> define un mapa de imagen, que puede contener áreas de enlace que se activan al hacer clic sobre una imagen.
