Diseño Web Responsivo

El diseño web responsivo permite que los sitios se adapten a diferentes tamaños de pantalla, mejorando la experiencia del usuario en múltiples dispositivos.

Viewport

El viewport es la ventana visible del contenido en el navegador del dispositivo. Ayuda a que el contenido se ajuste adecuadamente a diferentes pantallas.

Media Queries

Las media queries permiten aplicar estilos CSS diferentes en función del tamaño de pantalla.

Bootstrap 5

Bootstrap es un framework CSS que facilita el desarrollo de sitios web responsivos mediante un sistema de grid, contenedores y componentes reutilizables.
