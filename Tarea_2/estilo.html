Static

Este es el valor predeterminado para los elementos. El elemento se coloca según el flujo normal del documento.
Las propiedades:

  • left
  • right
  • top
  • bottom
  • z-index
No afectan a un elemento con posicion: static

Relative

Los elementos con posicion: relative permanecen en el flujo normal del documento.
Pero, a diferencia de los elementos estáticos, las propiedades:

  • left
  • right
  • top
  • bottom
  • z-index
Sí afectan la posición del elemento.
Se aplica al elemento un desplazamiento, basado en los valores de las propiedades:
  • left
  • right
  • top
  • bottom
en relación con sí mismo.

Absolute

Los elementos con posicion: absoluta se colocan en relación con sus elementos principales.
No se crea ningún espacio para el elemento en el diseño de página. Los valores de:

  • left
  • right
  • top
  • bottom
determinan la posición final del elemento.

Fixed

Los elementos de posición fija son similares a los elementos de posición absoluta.
También se eliminan del flujo normal del documento.
Pero a diferencia de los elementos absolutamente posicionados, siempre están posicionados en relación con el elemento <html>.

Z-index

La propiedad del índice z especifica el orden de pila de un elemento.
Un elemento con un orden de pila mayor siempre está delante de un elemento con un orden de pila menor.
